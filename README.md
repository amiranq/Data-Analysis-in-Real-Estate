# README

## Описание проекта

Блокнот `real_estate_eda_regression.ipynb` содержит анализ набора данных недвижимости и построение регрессионных моделей для оценки стоимости жилья за единицу площади.

Основные этапы работы:
1. загрузка и подготовка данных;
2. построение диаграмм признаков набора данных;
3. визуализация распределений и зависимостей;
4. построение базовой множественной линейной регрессии;
5. улучшение модели за счет новых признаков;
6. сравнение качества базовой и итоговой моделей.

## Структура блокнота

### 1. Подготовка окружения

- установка дополнительных библиотек;
- импорт библиотек;
- настройка отображения графиков;
- загрузка исходного датасета.

### 2. Описательный анализ данных

- распределение даты сделки;
- распределение возраста дома;
- распределение расстояния до метро;
- распределение количества магазинов рядом;
- распределение объектов на карте.

### 3. Анализ зависимости цены от признаков

- зависимость цены от количества магазинов;
- зависимость цены от расстояния до метро;
- зависимость цены от возраста дома;
- визуализация географического положения объектов.

### 4. Построение регрессионных моделей

- базовая множественная линейная регрессия;
- проверка значимости признаков через OLS;
- добавление признака `central_cluster`;
- логарифмирование расстояния до метро;
- добавление признака `house_age_sq`;
- сравнение базовой и итоговой моделей.

## Итог

В результате построена итоговая регрессионная модель, учитывающая расстояние до метро, пространственное положение объекта, возраст дома, количество магазинов поблизости и нелинейные зависимости между признаками.
